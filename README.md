# Currency MCP Agent

## Кратко
Система на базе MCP для получения курсов валют, конвертации и вызова внешних инструментов через единый слой оркестрации. Репозиторий разделяет сервер инструментов и слой агента, что делает поведение системы более прозрачным и удобным для тестирования.

## Задача
Нужна система, которая умеет вызывать внешние инструменты для получения курсов валют и конвертации значений, при этом логика принятия решения о вызове должна быть отделена от самой реализации инструментов. Для этого используется схема с сервером инструментов и отдельным слоем агента.

## Что улучшено
- инструменты вынесены в отдельный сервер;
- логика принятия решения о вызове инструментов отделена от реализации самих инструментов;
- структура проекта стала более пригодной для тестирования, локального запуска и контейнеризации.

## Архитектура
```text
[запрос пользователя]
          |
          v
      [агент]
          |
          v
[решение о вызове инструмента]
          |
          v
 [сервер инструментов MCP]
          |
          v
[получение курса / конвертация]
          |
          v
 [ответ пользователю]
```

## Структура репозитория
```text
agent/       логика агента
mcp_server/  сервер инструментов
config/      конфигурация
tests/       тесты
```

## Критерии качества
- корректность вызова инструментов;
- корректность вычисления конвертации;
- устойчивость поведения при ошибках внешнего источника;
- воспроизводимость локального и контейнерного запуска.

## Ключевой вывод
Разделение системы на слой агента и сервер инструментов делает её прозрачнее, упрощает тестирование и снижает связанность компонентов. Такой подход удобнее поддерживать, чем схему, где вся логика встроена в один слой.

## Запуск
```bash
docker compose up --build
```

## Ограничения
- итоговое качество зависит от надёжности внешнего источника данных;
- при расширении числа инструментов потребуется более строгая политика маршрутизации вызовов;
- для рабочего контура нужны ограничения по ошибкам, тайм-аутам и повторным попыткам.
