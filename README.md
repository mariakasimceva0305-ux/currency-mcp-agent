# Currency MCP Agent

Локальный сервис для работы с валютными курсами и конвертацией через `MCP` и агентный слой. Решение разделяет доступ к инструментам и логику обработки пользовательского запроса.

## Задача
Нужно принимать пользовательский запрос, определить нужный инструмент, получить внешний или локальный результат и вернуть структурированный ответ.

## Архитектура решения
```text
Пользовательский запрос
          │
          ▼
Агентный слой
          │
          ▼
MCP-сервер инструментов
          │
    ┌─────┴─────┐
    ▼           ▼
Курсы валют   Конвертация
    │           │
    └─────┬─────┘
          ▼
Структурированный ответ
```

## Структура репозитория
- `agent/` — логика принятия решения и маршрутизации запроса;
- `mcp_server/` — реализация инструментов и их интерфейсов;
- `config/` — настройки;
- `tests/` — проверка сценариев;
- `docker-compose.yml` — локальный запуск;
- `Dockerfile` — сборка контейнера.

## Поддерживаемые сценарии
- получение актуального курса;
- конвертация суммы между валютами;
- вызов инструментов через единый интерфейс.

## Метрики
- корректность выполнения сценариев;
- доля успешных вызовов инструментов;
- задержка ответа;
- покрытие тестами основных пользовательских маршрутов.

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
docker compose up --build
```

## Ограничения и развитие
- Качество ответа зависит от надёжности источника курсов;
- для расширения набора инструментов нужен единый контракт параметров и ответов;
- следующий шаг — добавить журналирование вызовов и формальную проверку ответов инструментов.
